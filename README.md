Deskripsi Proyek
Aplikasi e-commerce sederhana dengan fitur login, daftar produk, keranjang belanja, checkout, dan logout. Menggunakan session untuk autentikasi dan database MySQL untuk penyimpanan data.

Fitur Utama
Login/Register: Autentikasi pengguna.
Dashboard: Halaman utama dengan link ke produk.
Daftar Produk: Grid produk dengan card yang bagus.
Keranjang Belanja: Tambah, update, hapus item.
Checkout: Proses pembayaran dan konfirmasi sukses.
Responsive: Desain yang responsif untuk mobile dan desktop.

Cara Menjalankan Aplikasi
1. Jalankan XAMPP: Start Apache dan MySQL.
2. Akses Aplikasi: Buka browser dan pergi ke http://localhost/ecommerce/index.php.
3. Daftar Akun: Klik "Daftar" di login, isi username dan password.
4. Login: Masuk dengan akun yang dibuat.
Navigasi:
1. Dashboard: Klik "Lihat Daftar Produk".
2. Produk: Tambah ke keranjang, ubah quantity.
3. Keranjang: Update quantity, hapus item, checkout.
4. Checkout: Isi form, bayar, lihat sukses.
5. Logout: Klik logout di dashboard.
